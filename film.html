Film Analysis

This project analyzes multiple aspects of films from 1927-2010. The main purpose of this project is to analyze the genres, IMDb ratings and gender breakdown of each film in the dataset.

  • Genre and IMDB Ratings
  • Genre Analysis
  • Dialogue Analysis

Genre and IMDB Ratings Dashboard

This dashboard contains two graphs; one contains the genres of each film and the other contains the IMDb rating of each film. You can use the controls on the right to select a film or search for a specific film or year using the search box.

Genre Analysis Dashboard

This dashboard analyzes genres, as it contains a stacked bar graph of all the film genres throughout the years. It also has a bubble chart which breaks down the genres with the highest average IMDb rating. You can use the tools to the right to filter through genres, compare different years, and search for specific genres.

Dialogue Analysis Dashboard

This dashboard contains four graphs which analyze the word count broken down by gender for each film, as well as show which actors/ actresses appeared in the film. You can use the tools to the right filter through films, gender, year, and word count. You can also search for or highlight a movie that you want to find within the dataset.
