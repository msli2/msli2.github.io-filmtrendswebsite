SUBTITLE ANALYSIS

This project analyzes the subtitles from the top 50 grossing films of 2019. The main focus of this project is to show the overall sentiment of each film, as well as the frequency of the most common words in each film.

SENTIMENT ANALYSIS WORD ANALYSIS STATE ANALYSIS COUNTRY ANALYSIS

SENTIMENT ANALYSIS DASHBOARD

This dashboard shows the sentiment analysis of each film. You can use the navigation tools to the right to filter between the sentiment types, pick a film to look at, or search for a film or word in the search box.

WORD ANALYSIS DASHBOARD

This dashboard contains a word cloud and a bubble chart. Both can be used to look at the most common words which are said in a film, as well as the sentiment of each word. You can use the tools to the right to filter between sentiment types and the sum of the word counts. You can also search for a film to see the most frequently said words within the film.

STATE ANALYSIS DASHBOARD

This dashboard contains an ordered list of the most mentioned states from the subtitle data. There is also a bubble chart and a map which can be interacted with using the tools to the right.

COUNTRY ANALYSIS DASHBOARD

This dashboard contains an ordered list of the most mentioned countries from the subtitle data. There is also a bubble chart and a map which can be interacted with using the tools to the right.
